
# autobackup-ssh

A website data backup script via ssh.

### Requirements

- [Paramiko](http://www.paramiko.org/)

### Usage

```shell
$ pip install -r project/requirements.txt
$ cp config.sample.json config.json
$ python project/main.py config.json
```
